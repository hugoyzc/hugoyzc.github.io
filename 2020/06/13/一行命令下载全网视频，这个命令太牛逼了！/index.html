一行命令下载全网视频，这个命令太牛逼了！
发表于 2020-06-13| 更新于 2020-06-13|PC
字数总计:709|阅读时长: 2 分钟
| 阅读量:

Python

相信大家有遇到这样的情况，在网站上看到一些不错的视频，想要下载下来，但是没有找到网站的下载入口。

虽然像优酷、爱奇艺、腾讯视频都有自己的客户端，但下载的视频都是它们专用格式，其它播放器无法播放。

今天呢，Hugo 就给大家介绍一种高逼格的方法，只需要一行命令即可下载网站视频。

其实大家应该也不会陌生了，需要使用到的是编程软件Python，今天就向大家介绍一下Python的实际用途。很多人一听到编程可能就会感觉很难，其实不然，只需要安装Python环境，哪怕是小白也能轻松学会。

这款命令行工具叫You-Get，该工具基于一个Python的平台，所以我们需要先安装该平台。

安装教程

1、下载安装Python，文末给大家提供了3.8.3版本，下载后直接运行打开，特别注意需要勾选界面下方Add Python 3.8 to PATH ，接着点击 Install Now。

2、Python安装完毕后，接着我们需要把You-Get工具通过该平台运行到我们的电脑。

3、右击左下角开始点击运行或按下键盘Win+R组合键，输入cmd后回车。

4、输入命令并执行pip3 install you-get，如下图加载到100%即为成功。我们已经就将该工具安装到我们的电脑中，安装过程已经完毕，接下来是使用过程。

使用教程

以上安装过程只需安装一次，之后就可以调用下载使用了，接下来是使用过程。

同样Win+R输入cmd进入命令窗口，输入you-get 视频地址并回车。

示例：
you-get https://www.bilibili.com/video/BV16Z4y1s7ho

在粘贴视频下载的时候可以看到视频下载路径，即视频下载所在位置，当然也可以进行更改。

Hugo 测试了哔哩哔哩、优酷、腾讯视频都是可以下载的，根据官方介绍you-get支持的网页非常之多，包括大部分国内外主流媒体网站，除了可以主流视频网站，还可以下载音频、图片。

支持的网站

如果出现you-get 一段时间后无法下载视频？因为视频网站一直在更新，这时you-get也需要更新，输入代码pip3 install -u you-get即可更新工具。
