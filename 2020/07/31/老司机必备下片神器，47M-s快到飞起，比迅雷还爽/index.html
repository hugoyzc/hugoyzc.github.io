老司机必备下片神器，47M/s快到飞起，比迅雷还爽
发表于 2020-07-31| 更新于 2020-08-01|PC
字数总计:1.6k|阅读时长: 4 分钟

FDM

在下载工具这一块，大伙肯定第一时间想到迅雷。

迅雷是目前国民级下载工具，Hugo 还是从迅雷快车并驾齐驱的时代用过来的，不过在某些场景下还是不够好用。

对下载工具比较有了解的小伙伴还会用IDM，Hugo 目前也在用IDM，凭借多线程下载的优势，在下载文件时会比迅雷更快。

往期分享【IDM】【NDM】

还是因为多线程的优势，前段时间还被用来高速下载度盘文件，被称为「度盘神器」。

但是嘞，现在IDM的价格是149元，虽然是终身买断制，但相信很多朋友还是不愿意花着钱来买一个下载工具。

没关系，代替品还是有的嘛，其实除了IDM，还有一个同类工具叫FDM（Free Download Manager）。

FDM在国内可能不太出名，但在国外已经很有名了。

优点多得数不过来，无广告，界面简洁清爽，支持 BT、FTP 下载，支持批量下载、断点续传、捕获 HTTP 链接、FTP 目录浏览等等……

当然啦，最爽的还是跨平台，在很早开始就支持支持 Windows 与 macOS，最近还增加了安卓版！

话不多说，Hugo 带你们上车。

Windows版

FDM的界面有以前的快车下载软件内味，毕竟也是老牌下载工具了。

不过 Hugo 觉得这样的设计反而更加简单易懂。

界面并不是衡量下载工具好不好用的标准，下载功能好不好用才是真理。

首先是普通的HTTP链接下载，以下载QQ为例，FDM可以达到5.2MB/s。

然鹅，Hugo 用QQ浏览器自带的下载器下载时，下载速度在2MB/s左右波动。

一对比就能看出来，业余跟专业还是有差距的。

除此以外，FDM和IDM、NDM一样有浏览器插件，这是个好东西。

为啥Hugo 这么说呢，因为安装了浏览器插件后，配合【网盘助手】就可以破度盘限速下载文件。

经过 Hugo 测试，下载速度大概在800KB/s - 1.1MB/s浮动，相比官方客户端的100k还是好很多。

浏览器安装【网盘助手】和【FDM插件】后，登录度盘勾选需要下载的文件 – 生成链接 – 右键选择使用FDM下载。

具体的【网盘助手】和【FDM插件】安装步骤就不细说了。

因为之前的NDM文章中，Hugo 有详细介绍过插件安装教程，有兴趣的小伙伴点 这里。

如果度盘调用下载失败，请修改应用ID为778750。

FDM还支持直接下载Youtube和Fackbook的视频，把网页链接复制到这里就阔以下载，掌握了魔法的小伙伴可以试试。

安卓版

Hugo 本以为FDM不会有手机版本，结果前段时间居然推出了安卓版和Linux版。

跨平台的下载工具本来就不多，免费无广告还支持多协议下载的估计一个手指头都数得过来。。。

安卓端延续了简洁的设计风格，核心功能也被保留了下来，作为国外的下载工具，全平台都支持中文好评！

先测试下普通文件下载的速度好了。

emmm，居然刚添加一个QQ的下载链接就下载完了。。。

瞬息万变之间，Hugo 看到下载速度是47MB/s，是完全把宽带榨干的水平了。

当然，下载工具只能做到最大化下载速度，而最终下载速度如何还是取决于文件存储服务器的速度。

除了普通的HTTP/FTP文件下载，安卓端的FDM也保留了BT下载，不过 Hugo 要先给大家打个预防针，一般这样的下载工具对国内BT的支持很差。

BT本质上是P2P下载，需要用户之间互相加速，所以种子越热门，下载速度越快。

有些种子明明是热门种子，却还是下载没速度，这就可能是tracker的问题了。

tracker是指运行于服务器上的一个程序，这个程序能够追踪到底有多少人同时在下载同一个文件。客户端连上tracker服务器，就会获得一个下载人员的名单，根据这个，BT会自动连上别人的机器进行下载。它是提供bt的服务器。把文件用bt发布出来的人需要知道该使用哪个服务器来为要发布的文件提供tracker。

大多数BT下载工具都很依赖tracker来追踪到其他用户，这样才得以把下载速度提上来。

迅雷最厉害之处就是有自家的缓存服务器，资源量巨多，即使是没人下载的种子（死种）也能通过服务器加速下载。

当然，要会员的，没钱的散了吧。

随便以一个电影种子为例，在添加tracker之前，下载速度慢得可怜，14KB/s。。。

即使添加了tracker后，下载速度得到了提升，但也只是在78KB/s左右浮动。

所以嚯，同一个种子，毕竟国内的种子还是只有国内的开发者能懂得怎么下载。

最后

FDM作为一个老牌下载工具，还获得过很多奖项，如今还推出了安卓版本。

虽然在BT下载上比较弱，但优势在于够简洁稳定，并且在普通文件的下载速度上，亲测优于迅雷。

而PC端和安卓端各有优势，PC端配合插件除了能高速下载普通文件，还能高速下载度盘文件。
