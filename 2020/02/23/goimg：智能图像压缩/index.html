智能图像压缩

为图像文件瘦身，让你的图片加载飞快！ 图片压缩速度确实挺快，不需要做多余设置，只需要选择文件上传 goimg 会根据深度学习引擎自动压缩图片为你自动压缩图片，然后点击下载即可。

https://goimg.io/

极致的压缩效果

使用聪明的有损压缩技术来减少你的 PNG、JPG、GIF 文件的存储大小。通过深度学习的图像识别，来选择性地减少图像中的颜色数，在减少图片体积达到极限的同时，让肉眼几乎看不出与原图的差异！

如何压缩图片？

把需要压缩的图片文件拖放至屏幕任意位置，即可上传压缩图片。压缩完成后将文件存入计算机或你的移动设备中，全部过程仅此 1 步。
压缩品质

Goimg 的图片压缩引擎能够将图片文件的字节减少 75% 左右的平均值，而且肉眼几乎无法看出与原图的差异，质量及文件大小都堪称完美。

文件的安全和隐私

当你的图片压缩完成的 60 分钟后，将会永久从 Goimg 网站的服务器删除。同时文件链接即失效，100% 保障你的隐私。
支持所有操作系统

使用用聪明的有损压缩算法，基于图形识别和深度学习，来选择性的进行压缩，不对图片进行格式破坏，与原图保持一致的适用性。

云端压缩引擎

Goimg 基于浏览器运行，整个压缩过程在服务器端进行，不会降低计算机的运行速度。只要保证你的计算机有基础的网络连接状态即可。

最适合谁用？

UI 设计师、平面设计师、客户端 / 前端开发工程师必备！编辑、摄影师、网店、个人博客… 一切完美主义注重细节品质的创作者。
